O que é HTML

Voltar

Criado por Tim Berners-Lee, o HTML é uma Linguagem de Marcação de Hipertexto, e sua função é delimitar/marcar estruturar uma página web e seu conteúdo, através de TAGS de forma com que eles sejam compreendidos tanto pela máquina, quanto pelo usuário.

Em outras palavras é você dar instruções para máquina, falando a mesma linguagem que ela, sendo assim ela pode devolver essas instruções de maneira compreensível para o usuário.

Através dessa linguagem HTML, podemos criar páginas com textos contendo titúlos, parágrafos, listas e etc.

Princípios básicos para construir uma página HTML

Voltar

Para construção de uma página é necessário ter:

  1. Conhecimento das principais Tags HTML;
  2. Um editor de texto como VsCode, Sublime Text ou Notepad++;
  3. Boas prática como acessividade e responsividade;

Quando o desenvolvedor constrói uma página considerando estes princípios básicos, ele consegue criar uma página onde usuário terá uma interação confortável e amigável ao navegar por ela.

O desenvolvedor deve saber utilizar de outros artifícios como as cores certas, fonte ideal e manter um visual clean da página, tudo para que os seus usuários tenham uma experiência positiva durante a navegabilidade.

A finalidade da página irá determinar o tamanho, peso e tipo da fonte a ser utilizada.

As mídias como: Imagens, áudio, video e etc.

E por último e mais importante a semântica, pois esta, embora não seja visível para o usuário, é ela que faz a diferença para buscadores e leitores de tela, tornando-se assim uma ótima página para pessoas que necessitam do uso de acessiblidade.
